# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please report it to:
**security@aquilesindustries.com**

For Code of Conduct reports, see CODE_OF_CONDUCT.md (conduct@aquilesindustries.com).

Please include:
- a clear description of the issue
- steps to reproduce
- affected components and versions (if known)

We aim to respond within 72 hours.

## Supported Versions

This repository publishes documentation and governance for the Aquiles Faro
module. Security fixes will be documented in the repository history and
changelog notes (if applicable).

## Responsible Disclosure

Please do not disclose vulnerabilities publicly until we have had a chance to
review and respond.
