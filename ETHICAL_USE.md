# Ethical Use — Aquiles

Este documento no es una licencia: es una guía comunitaria para uso responsable.

## Principios
- **Transparencia:** explicar qué hace el sistema y cómo se usa.
- **Consentimiento:** no operar sobre datos ajenos sin autorización.
- **Minimización de datos:** recolectar lo mínimo necesario.
- **Auditabilidad:** mantener evidencia y trazabilidad.
- **No daño:** evitar automatización opaca o decisiones sin control humano.

## Ejemplos de uso no recomendado
- Scraping masivo sin consentimiento.
- Publicación de datos personales o sensibles.
- Automatización que oculte impactos o errores.

Si tienes dudas sobre un caso de uso, abre un issue para discutirlo.
