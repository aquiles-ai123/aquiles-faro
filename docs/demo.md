# Demo pública de Aquiles Faro

## Propósito

Esta demo pública muestra cómo Faro puede enlazar documentación abierta con una corrida real y
reproducible de Aquiles.

## Qué incluye

- una nota pública de la corrida: `public/aquiles_reasoning_demo_2026-03-13.md`
- un manifiesto JSON resumido: `public/aquiles_reasoning_demo_manifest_2026-03-13.json`
- un artifact descargable con la corrida: `public/demo_pack/aquiles_reasoning_demo_2026-03-13.tar.gz`
- checksum verificable: `public/demo_pack/aquiles_reasoning_demo_2026-03-13.sha256`

## Qué demuestra

La corrida pública demuestra que Aquiles ya puede:

- descomponer claims de distinta clase
- inferir contexto, relación y objeto
- dejar un rastro metodológico visible
- reconocer límites cuando la evidencia trazable todavía no alcanza

## Qué no demuestra todavía

- soporte fuerte y consistente de retrieval trazable por dominio
- una capa cerrada de verificación final
- una postura de producción endurecida

## Cómo leer la demo

1. revisar `public/aquiles_reasoning_demo_2026-03-13.md`
2. abrir el manifest JSON
3. descargar el artifact y verificar el hash
4. contrastar resultados con el estado metodológico del proyecto
